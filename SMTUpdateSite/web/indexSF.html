SMT-LIB resources

This page lists a number of resources about SMT solvers and the SMT-LIB language.

  • A tutorial about the SMT-LIB language.
  • A software application and library that accompanies the tutorial.
  • A User Guide for the application, library, and plug-in.
  • An Eclipse plug-in site that is a wrapper for the jSMTLIB library, creating a GUI tool for using SMT-LIB and SMT solvers.
Support for some of the development of these resources was provided by GrammaTech, Inc., a user of SMT tools and a creator and vendor of program analysis research and tools.
