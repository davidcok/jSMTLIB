SMT-LIB resources

GrammaTech uses SMT solvers as part of its suite of sophisticated static analysis tools. This page lists a number of resources about SMT solvers and the SMT-LIB language. These are not GrammaTech products; they are linked here solely for convenience and as a service. There is no charge for their use for educational or research purposes (but they may be copyrighted and there may be restrictions on commercial use).

  • A tutorial about the SMT-LIB language.
  • A software application and library that accompanies the tutorial.
  • A User Guide for the application, library, and plug-in.
  • An Eclipse plug-in site that is a wrapper for the jSMTLIB library, creating a GUI tool for using SMT-LIB and SMT solvers.
